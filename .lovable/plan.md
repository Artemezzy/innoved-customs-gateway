
# План: Секция отзывов клиентов на главной странице

## Обзор
Добавление секции с отзывами клиентов в виде слайдера на главную страницу после блока "О компании". Будет использован уже существующий компонент карусели (Embla Carousel) и стилистика проекта.

## Структура отзывов
Каждый отзыв будет содержать:
- Аватар клиента (инициалы)
- Имя и должность/компания
- Текст отзыва
- Рейтинг (звёзды)

## Технические детали

### Дизайн карточки отзыва
```text
┌─────────────────────────────────┐
│  ★★★★★                          │
│                                 │
│  "Текст отзыва клиента..."      │
│                                 │
│  ┌──┐  Имя Фамилия             │
│  │АМ│  Должность, Компания     │
│  └──┘                          │
└─────────────────────────────────┘
```

### Стилизация
- Фон секции: `bg-background` (светлый, контраст с About)
- Карточки: `bg-card` с `shadow-card`
- Рейтинг: звёзды желтого цвета (`text-yellow-400`)
- Аватар: круглый с инициалами и `bg-primary`
- Анимация появления: `animate-fade-in`

### Адаптивность
- Мобильные: 1 отзыв на слайд
- Планшеты: 1 отзыв на слайд  
- Десктоп: 1 отзыв на слайд (центрированный)

### Автопрокрутка
- Интервал: 5 секунд
- Использование Embla Carousel Autoplay plugin
- Остановка при наведении мыши
